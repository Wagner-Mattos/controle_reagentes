# Requisitos Funcionais

# 🎯 Visão Geral

Este documento descreve os requisitos funcionais do Sistema de Controle de Reagentes.

Os requisitos funcionais representam as funcionalidades que o sistema deverá oferecer aos usuários.

---

# 🔐 RF001 - Autenticação

O sistema deverá permitir:

- Login
- Logout
- Controle de sessão
- Recuperação futura de senha

---

# 👤 RF002 - Controle de Usuários

O sistema deverá permitir:

- Cadastro de usuários
- Edição de usuários
- Desativação de usuários
- Controle de permissões

Perfis previstos:

- admin
- supervisor
- technician
- viewer

---

# 🧪 RF003 - Cadastro de Reagentes

O sistema deverá permitir:

- Criar reagentes
- Editar reagentes
- Remover reagentes
- Consultar reagentes

Campos mínimos:

- Nome
- CAS
- Lote
- Quantidade
- Unidade
- Validade

---

# 📦 RF004 - Controle de Estoque

O sistema deverá:

- Atualizar estoque automaticamente
- Impedir estoque negativo
- Registrar histórico

---

# 🔄 RF005 - Movimentações

O sistema deverá permitir:

- Entrada
- Saída
- Ajuste
- Descarte
- Transferência

---

# 📍 RF006 - Controle de Localização

O sistema deverá permitir:

- Cadastro de laboratórios
- Cadastro de armários
- Associação de reagentes

---

# 🏢 RF007 - Controle de Fornecedores

O sistema deverá permitir:

- Cadastro de fornecedores
- Consulta
- Edição

---

# 📄 RF008 - Upload de Arquivos

O sistema deverá permitir upload de:

- FISPQ/SDS
- Fotos
- Documentos técnicos

---

# 🔍 RF009 - Busca e Filtros

O sistema deverá permitir:

- Busca por nome
- Busca por CAS
- Filtros por validade
- Filtros por localização

---

# 📊 RF010 - Dashboard

O sistema deverá apresentar:

- Estoque geral
- Produtos vencidos
- Produtos próximos do vencimento
- Movimentações recentes

---

# 🚨 RF011 - Alertas

O sistema deverá alertar:

- Estoque baixo
- Produto vencido
- Validade próxima

---

# 🔒 RF012 - Controle de Permissões

O sistema deverá restringir ações por perfil.

---

# 🧾 RF013 - Auditoria

O sistema deverá registrar:

- Login
- Alterações
- Exclusões
- Movimentações

---

# 🔄 RF014 - Realtime

O sistema deverá atualizar informações em tempo real.

---

# 📱 RF015 - Responsividade

O sistema deverá funcionar em:

- Desktop
- Tablet
- Smartphone

---

# 🧪 RF016 - Inventário

O sistema deverá permitir:

- Conferência de estoque
- Ajustes
- Registro de inventário

---

# 🖨️ RF017 - Etiquetas

Planejado futuramente:

- QRCode
- Código de barras
- Impressão de etiquetas

---

# 🔬 RF018 - Integrações Futuras

Planejado:

- Equipamentos laboratoriais
- RFID
- Sensores ambientais

---

# 🎯 Objetivo Final

Construir um sistema:

- Seguro
- Auditável
- Simples
- Escalável
- Fácil de operar
